新規プロジェクトまたは既存プロジェクトのClaude Code環境を初期化する。

プロジェクトタイプ: $ARGUMENTS
（例: frontend, backend, fullstack, 空欄の場合は自動判定）

## 手順

### 1. プロジェクト分析
- README.md、package.json、設定ファイルを読み込み
- 使用言語・フレームワークを特定
- ディレクトリ構造を把握

### 2. CLAUDE.md の生成
以下の情報を含むCLAUDE.mdを作成:

```markdown
# プロジェクト名

## 概要
プロジェクトの説明

## 技術スタック
- 言語:
- フレームワーク:
- 主要ライブラリ:

## ディレクトリ構造
主要ディレクトリの説明

## 開発コマンド
- 開発サーバー起動:
- テスト実行:
- ビルド:
- Lint:

## コーディング規約
- このプロジェクト固有のルール

## 注意事項
- 既知の問題や制約
```

### 3. .claude/commands/ の設定
プロジェクトに適したカスタムコマンドを提案

### 4. .claudeignore の作成
除外すべきディレクトリ・ファイルを設定:
- node_modules/
- dist/
- .env*
- *.log

## 注意事項

- 既存のCLAUDE.mdがある場合は上書きせず、マージを提案
- 機密情報は含めない
